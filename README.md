# Codiert
For start:

You need a .png file. Name it "ineedpass". Put it near project and start GPK.py
```
python "path/to/file/GPK.py"
```
After that, a new file with the same image will appear, but it contains your personal encryption password.

To encrypt or decrypt, you must run logic.py Insert path to password-image and with commands "enc ***" or "dec ***" successfully encrypt the necessary data
```
python "path/to/file/logic.py"
path/to/file/nothing.png
->>enc avik
gAAAAABplLadcm4_n_CB9dTpEXlekyXaI-EgiPPHFGUq194U4BeJuZalVsuG8FZe5RiJXGp_wf3aJzPVKuUcbBO6AtQD1pXX5g==
->>dec gAAAAABplLadcm4_n_CB9dTpEXlekyXaI-EgiPPHFGUq194U4BeJuZalVsuG8FZe5RiJXGp_wf3aJzPVKuUcbBO6AtQD1pXX5g==
avik
```
For exit: write 000
